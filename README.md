# 📊 HR Analytics Dashboard (Power BI)

## 🚀 Project Overview

This project is an **HR Analytics Dashboard** built using Power BI.
It helps in analyzing employee data to understand **attrition trends, employee demographics, and salary insights**.

The dashboard provides meaningful insights for HR teams to take **data-driven decisions**.

---

## 📁 Dataset

The dataset contains employee-related information such as:

* Employee ID
* Age
* Salary
* Education
* Job Role
* Gender
* Attrition Status

---

## 📌 Key Metrics (KPIs)

* 👨‍💼 Total Employees: **1473**
* ❌ Attrition Count: **237**
* 📉 Attrition Rate: **16.1%**
* 🎂 Average Age: **36.92**
* 💰 Average Salary: **6.50K**
* ⏳ Average Years: **7.00**

---

## 📊 Dashboard Features

### 🔹 Attrition Analysis

* Attrition by Education
* Attrition by Age Group
* Attrition by Salary Slab
* Attrition by Job Role

### 🔹 Employee Insights

* Gender Distribution
* Department Wise Distribution
* Job Role Breakdown

### 🔹 Trend Analysis

* Attrition Trend Over Time

---

## 📈 Key Insights

* Employees aged **26–35** show highest attrition
* Most attrition happens in **low salary range (Up to 5k)**
* **Life Sciences** has the highest attrition among education fields
* Certain job roles like **Laboratory Technician & Sales Executive** show higher attrition
